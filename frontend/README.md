# 🚛 Agil – Gestion de Parc Automobile

Application web complète pour la gestion intelligente d’un parc automobile.

## ✨ Fonctionnalités

- **Dashboard** interactif avec graphiques 3D et données temps réel
- **Gestion des véhicules** (CRUD, suivi kilométrage, documents)
- **Gestion des missions** (planification, feuille de route)
- **Gestion des chauffeurs** (congés, performances)
- **Maintenance préventive** (alerte tous les 10 000 km)
- **Documents & IA** (OCR pour lecture automatique des amendes)
- **Suivi carburant** (consommations, cartes, alertes)
- **Statistiques avancées** (rapports, exports CSV)
- **Administration** (gestion des utilisateurs, rôles)
- **Paramètres utilisateur** (profil, notifications)

## 🛠️ Technologies

- **Frontend** : React 18, Vite, Tailwind CSS
- **Graphiques** : Recharts
- **Icônes** : Lucide React
- **Routage** : React Router DOM
